# SPX-OS v0.1

Операційна система суб'єктних агентів SPX (MVP).
Відповідає пакету специфікацій v0.1 (Manifest, File Structure, Bootstrap, Contracts, MWE).

## Швидкий старт
```bash
python -m venv .venv && source .venv/bin/activate  # або .\.venv\Scripts\activate на Windows
pip install -r requirements.txt
python main.py
```
Тести (базові скелети):
```bash
pytest -q
```
